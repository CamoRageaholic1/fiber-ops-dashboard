# Google Sheets Setup Guide

Complete guide for setting up Google Sheets integration with Fiber Ops Dashboard.

## Quick Start

1. Create Google Cloud Project
2. Enable Google Sheets API
3. Create Service Account
4. Download credentials.json
5. Share your sheet with service account
6. Add Sheet ID to .env

## Detailed Steps

See full documentation at: https://github.com/CamoRageaholic1/fiber-ops-dashboard

## Required Sheet Format

| Project Name | Total Footage | Completed Footage | Material Cost | Labor Cost | Total Cost | Date |
|--------------|---------------|-------------------|---------------|------------|------------|------|

All column names must match exactly (case-sensitive).
